- Todo documento precisa começar com <'!DOCTYPE html'>, para o navegador entender que o documento está em html;

- Deve também conter a tag <'html'>;

- "charset" é um atributo, e indica qual configuração de caracteres será utilizado;

- Usa-se <'title'> para armazenar informação não visiveis da pagina, cabeçalho, palavras-chaves para busca;

- A tag <'body> é literalmente o corpo do conteudo na nossa pagina;

- A tag <'p'> é usada para parágrafos!";

- O atributo "lang" é usado para definir em qual idioma estará a página;

- But, if an element is in another language, we must sinalize it!

- Existem alguns meta dados que podemos utilizar na tag meta, por exemplo:

author (autor do site), description (descrição do site),
